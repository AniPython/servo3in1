## DIY舵机学习套件3合1

使用 ESP32 做为开发板，micorpython 或 auduino 进行编程，wifi 远程遥控机器人  
特点和亮点：使用折叠纸板做为机器人的外壳，大降低了硬件成本

本项目包含了3个机器人：  
1. dog 小狗，使用2个舵机，是最简单的
